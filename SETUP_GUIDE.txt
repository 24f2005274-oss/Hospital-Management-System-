## 1. LOCAL SETUP

### Step 1: Extract and Navigate
```bash
# Extract the zip file
# Navigate to the project folder
cd Hospital-Management-System
```

### Step 2: Create Virtual Environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Run the Application
```bash
python app.py
```

### Step 5: Access the Application
```
http://localhost:5000
```

### Default Admin Login
- Username: admin
- Password: admin123

## 3. GITHUB SETUP & PUSH

### Step 1: Initialize Git Repository
```bash
# Navigate to project folder
cd Hospital-Management-System

# Initialize git
git init

# Check git status
git status
```

### Step 2: Add Files to Git
```bash
# Add all files
git add .

# Check what will be added
git status
```

### Step 3: Create First Commit
```bash
git commit -m "Initial commit: Hospital Management System"
```

### Step 4: Create GitHub Repository
1. Go to https://github.com/new
2. Enter repository name: `Hospital-Management-System`
3. Choose "Private" (recommended for assignment)
4. Click "Create repository"

### Step 5: Add Remote and Push
```bash
# Add remote (replace USERNAME with your GitHub username)
git remote add origin https://github.com/USERNAME/Hospital-Management-System.git

# Rename branch to main if needed
git branch -M main

# Push to GitHub
git push -u origin main
```

### Step 6: Verify Upload
1. Go to your GitHub repository URL
2. Refresh the page
3. You should see all project files

## 7. COMMON ISSUES & SOLUTIONS

### Issue: Python not found
```bash
# Check Python installation
python --version
# or
python3 --version
```

### Issue: pip command not found
```bash
# Use python -m pip instead
python -m pip install -r requirements.txt
```

### Issue: Port 5000 already in use
```bash
# Change port in app.py
app.run(debug=True, port=5001)
```

### Issue: Database locked
```bash
# Delete instance/hospital.db and restart
```

### Issue: Git not found
```bash
# Download from https://git-scm.com/
# Restart terminal after installation
```

### Issue: GitHub push fails
```bash
# Configure git first
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# Then try push again
git push -u origin main
```
