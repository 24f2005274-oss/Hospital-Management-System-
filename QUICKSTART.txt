🐛 TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════

Problem: "ModuleNotFoundError"
Solution: pip install -r requirements.txt

Problem: "Port 5000 already in use"
Solution: Edit app.py: app.run(debug=True, port=5001)

Problem: "Database error"
Solution: Delete instance/hospital.db, restart app

Problem: "Template not found"
Solution: Ensure templates/ folder structure is correct
